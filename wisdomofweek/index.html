To insert the wisdom of the week in your own email signature:

Most email clients do not allow HTML to be typed directly into the signature block if yours does or has the option to, then follow the HTML method. If your client does not allow HTML in the signature block then follow the Copy/Paste method.

HTML Method:

If your email client has the option to include HTML in the signature, then enable it and then copy and paste the following HTML code at the end of the signature section in your email client.
<a href="https://tugzrida.github.io/wisdomofweek/" target="_blank"><img src="https://tugzrida.github.io/wisdomofweek/wisdomofweek.png" width="540px"></a>

Copy/Paste Method:

Most email clients do not allow HTML directly in the signature block. For these email clients(eg. Apple Mail, iOS mail, Gmail, etc...):
  1. Open this window
  2. Select all the contents (Mac: ⌘ + A, Windows: CTRL + A, iOS: Tap + Hold ➔ Let Go ➔ Select All)
  3. Copy (Mac: ⌘ + C, Windows: CTRL + C, iOS: Tap Copy)
  4. Paste into the end of your signature block in your email client (Mac: ⌘ + V, Windows: CTRL + V, iOS: Tap + Hold ➔ Let Go ➔ Paste)
